# IP-Port
- ⛅ IP-Port 是一个批量快速给IP添加端口号的工具
- ⛅ 支持多个IP多个端口支持一键添加
- 📧 意见反馈：0xShe@t00ls.net

### 使用方法：
- 🍺 输入IP地址 
- 🍺 输入端口号用逗号隔开，如果有多个连续的端口号用-隔开，如8080-8090
- 🍺 例：80, 443, 8080, 8000-8005
